# 产品需求文档 (PRD)

| 项目名称 | 批量视频水印处理大师 (Batch Video Watermark Master) |
| :--- | :--- |
| **产品形态** | 桌面端软件 (支持 Windows / macOS 双系统) |
| **核心定位** | 面向短视频创作者的轻量化、本地化批量去水印与贴纸覆盖工具 |

---

## 1. 核心业务流程与逻辑

### 1.1 业务规则
* **同批次单一策略约束：** 用户单次导入的一批视频（3-5个），在执行处理时，**只能统一选择**【策略A：智能去水印】或【策略B：文字贴纸覆盖】，不支持在一批任务中混合处理。
* **本地化处理：** 所有视频的读取、解码、图像修复、编码均在用户本地完成，不消耗云端算力。

### 1.2 核心用户旅程 (User Journey)
1. 启动软件 
2. 批量拖入视频 (3-5个)
3. 框选水印区域 
4. 选择处理模式（去水印 或 选定一款贴纸样式） 
5. 点击“开始处理” 
6. 导出成品视频

---

## 2. 功能需求详情 (Functional Requirements)

### 2.1 模块一：视频导入与列表管理
* **需求描述：** 支持用户将需要处理的视频批量导入软件中。
* **功能点：**
    * **导入方式：** 支持点击按钮打开文件选择器（多选），或直接将视频文件/文件夹**拖拽**进软件指定区域。
    * **数量拦截：** 限制单次批量上传数量为 **3 - 5 个视频**。若超过5个，前端予以弹窗提示并拦截。
    * **列表展现：** 导入后以列表/网格形式展示视频名称、时长、分辨率及当前状态（等待中/处理中/已完成）。

### 2.2 模块二：交互框选与预览 (核心体验)
* **需求描述：** 用户选中列表中的第一个视频作为“模板视频”，在画面上框选出水印的固定位置。
* **功能点：**
    * **视频预览播放器：** 居中展示，支持暂停，方便用户定格含有水印的帧。
    * **Canvas 鼠标框选：** 用户可在视频画面上按住鼠标左键拖拽，画出一个**矩形高亮框 (ROI)**。
    * **坐标等比例映射（关键技术点）：** 
      > **注意：** 用户在软件界面上看到的视频是经过缩放的（例如 480P），但原视频可能是 1080P。前端必须计算出框选区域在原视频中的真实像素坐标 $(X, Y, W, H)$ 并传递给后端。

### 2.3 模块三：处理策略配置区 (二选一)
界面提供一个切换开关（Tabs 或 Radio Group），同一批任务只能选一项：

#### 【选项 A】智能去水印
* **功能点：** 勾选后，后端调用 OpenCV Inpainting 算法。对框选区域进行边缘像素向内插值修复，达到肉眼看去无明显违和感、无明显痕迹的效果。

#### 【选项 B】文字贴纸覆盖
* **功能点：** 
    * **文字输入：** 提供输入框，允许用户自定义贴纸文字（如“精彩视频”、“关注我”），字数限制在 20 字以内。
    * **样式选择（8选1）：** 提供 8 种预设样式供用户一键选择（界面提供对应小图预览）：
        1. *样式1（经典黑白）：* 黑底半透明背景 + 白色文字（圆角）。
        2. *样式2（综艺爆款）：* 黄色粗字体 + 黑色阴影描边（最不容易穿帮）。
        3. *样式3（警示红）：* 纯红文字，无背景。
        4. *样式4（商务风）：* 浅灰背景 + 深灰文字。
        5. *样式5 - 样式8：* 可根据开发视觉库灵活配置纯色、渐变或带轻量边框的样式。

### 2.4 模块四：异步批量处理引擎
* **需求描述：** 点击“开始处理”后，软件在后台逐个处理视频，界面不卡死。
* **功能点：**
    * **队列管理：** 按照列表顺序依次处理，界面实时更新总体进度条与单个视频进度条（0% - 100%）。
    * **后端执行：** Python 后端收到坐标和策略后，调用 FFmpeg/OpenCV 执行转码。
    * **文件输出：** 将处理完的视频输出到用户指定的本地目录（默认原视频同级目录下的 `_output` 文件夹）。

## 3. 非功能需求 (Non-Functional Requirements)

* **性能与响应：** 
    * 15秒~3分钟的视频，在常规办公电脑上，贴纸覆盖策略的处理速度应接近 **1:0.5**（即1分钟视频约30秒内处理完）。
    * 智能去水印由于涉及逐帧像素修复，速度略慢，需在界面上提供明确的“正在深度修复中，请稍候”的文案提示。
* **跨平台兼容性：** 
    * **Windows 端：** 支持 Win10 及以上系统（x64架构）。
    * **macOS 端：** 支持 macOS 11 及以上系统（兼容 Intel 芯片与 Apple Silicon M1/M2/M3 芯片）。
* **存储与清理：** 软件在转码过程中产生的临时图片帧（TS切片/临时图片），必须在单个视频处理完成后**立刻自动销毁**，严控本地磁盘占用。
